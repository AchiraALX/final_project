Auth Endpoints

Endpoint Methods Purpose
/auth POST Authenticates user for matching password and username
/auth/register GET, POST GET: Fetches the registration page
POST: Creates an account for new users
/auth/reset-password GET Initiates the password reset process
/auth/update-password/<reset-token> PUT Updates the user's password
/auth/login GET, POST GET: Gets the login page
POST: Logs in the user

API Endpoints

Endpoint Methods Purpose
/api GET Gets API home page
/api/user/<query> GET Gets a single user that matches the specified query
/api/users/<query> GET Gets all users that match the specified query
/api/all-blogs GET Gets all blogs available
/api/add-blog POST Creates a new blog
/api/blog/<query> GET Gets all available blogs that match the specified query
/api/blog-update/<key> PUT Updates the blog that matches the key
/api/user-update/<key> PUT Updates a user that matches the specified (key)
/api/delete-blog/<key> DELETE Deletes a blog that matches the specified key
/api/destroy-user/<key> DELETE Deletes a user that matches the specified key
